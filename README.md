## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/arafateasin/SimpleLottery.git
   cd react-lottery-game
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open the app in your browser at:
   ```text
   http://localhost:5173
   ```
